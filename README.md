# Reference

Personal reference pages, served at https://camerondutra.github.io

Plain static HTML, no build step. Edit a file, commit, push — Pages redeploys in about 30 seconds.

| File | What it is |
| --- | --- |
| `index.html` | Hub — links every page |
| `cards.html` | Credit card cheat sheet, by spending category |
| `robots.txt` | Asks search engines not to index. **Not access control** — this repo is public. |

## Adding a page

Drop the `.html` file in, add a matching `<a class="item">` block to `index.html`, push.

## Note on what goes here

This repo is public, and so is everything served from it. Anything with an account number, a balance, a credential, or a card number does not belong in this repo.
